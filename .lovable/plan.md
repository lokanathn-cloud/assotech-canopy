

# Assotech Canopy - Premium Real Estate Landing Page

## Project Overview
A high-converting, premium landing page for Assotech Canopy, a RERA-approved luxury plotted development in Crossing Republik, Ghaziabad. The design will emphasize the dual value proposition: **Legacy Builders** (end-users seeking dream homes) and **Wealth Maximizers** (investors seeking appreciation).

---

## Design System

### Color Palette (Nature-Inspired, Premium)
- **Hero Green**: #2D7A3E - Primary CTAs, accent elements
- **Earth Brown**: #6B5B47 - Secondary elements, grounding
- **Cream/Off-White**: #F9F7F3 - Primary backgrounds
- **Deep Charcoal**: #1F1F1F - Headlines, primary text
- **Sky Blue**: #87CEEB - Secondary CTAs
- **Gold**: #D4A574 - Premium highlights

### Typography
- **Headlines**: Poppins (Modern, Premium feel)
- **Body**: Open Sans (Highly readable)
- **Hierarchy**: H1 (48-56px) → H2 (32-40px) → H3 (24-28px) → Body (14-16px)
